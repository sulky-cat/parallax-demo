Parallax.js

Без событий!

Вместо событий прокрутки используется requestAnimationFrame, что добавляет плавности к прокрутке.

   <div data-prlx> Элемент <div>
<div>

Настройка нулевой позиции

"top" - Элемент в позиции (0,0), когда верхний край родителя соприкасается с верхним краем экрана

"center" - Элемент в позиции (0,0), когда центр родителя (по высоте) соприкасается с центром экрана (по высоте)

"bottom" - Элемент в позиции (0,0), когда ниэний край родителя соприкасается с нижним краем экрана

<div data-prlx-parent data-center="top">
   <div data-prlx> Элемент <div>
<div>
<div data-prlx-parent data-center="center">
   <div data-prlx> Элемент <div>
<div>
<div data-prlx-parent data-center="bottom">
   <div data-prlx> Элемент <div>
<div>

Методы

setEvents() - утсановить плагин

destroyEvents() - удалить плагин

Установить / удалить для всех параллаксов на странице

const parallaxPlug = new Parallax(document.querySelectorAll('[data-prlx-parent]'));
const buttons = document.querySelectorAll('[data-event-all]');
buttons.forEach(button => {
  button.addEventListener('click', () => {
    const data = button.dataset.eventAll;
    switch (data) {
      case "init":
        parallaxPlug.init();
        break;
      case "destroy":
        parallaxPlug.destroy();
        break;
      default:
        break;
    }
  });
});

Установить / удалить для одного параллакса

const parallaxPlug = new Parallax(document.querySelectorAll('[data-prlx-parent]'));
const jsParent = parallaxPlug.elements.find(el => el.parent.classList.contains('demo_js'));
const buttons = document.querySelectorAll('[data-event]');
buttons.forEach(button => {
  button.addEventListener('click', () => {
    const data = button.dataset.event;
    switch (data) {
      case "init":
        jsParent.init();
        break;
      case "destroy":
        jsParent.destroy();
        break;
      default:
        break;
    }
  });
});
